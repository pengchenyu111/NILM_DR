# 一、环境配置说明

1. 创建虚拟环境

   ```shell
   conda create -n nilmtk-ev python=3.7
   ```
   
2. 导入所需依赖

   ```shell
   # 注意需要进入requirements所在的文件夹
    conda install --yes --file requirements.txt
   ```
   
3. CSDN上的博客都是水的！直接将github上的nilmtk、nilm_metadata、nilmtk_contrib的相关文件夹复制过来即可(只需要子文件夹，不要带setup.py的那个父文件夹)。

   ```shell
   # 注意需要进入requirements所在的文件夹
    conda install --yes --file requirements.txt
   ```
